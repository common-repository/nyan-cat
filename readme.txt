The ultimate accessory to WordPress 3.2's Distraction-Free Writing mode. Get in the zone... the NYAN ZONE!

== Description ==

Let the hypnotic beat of [NYAN CAT](http://nyan.cat/) help you focus on your writing! The Nyan Cat plugin adds [NYAN CAT](http://nyan.cat/) to WordPress 3.2's "Distraction-Free Writing" fullscreen mode.

**WordPress 3.2 is required.**
